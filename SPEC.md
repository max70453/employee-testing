# Онлайн-система тестирования сотрудников водоканала

## 1. Обзор проекта

**Название:** WaterTest - система тестирования сотрудников водоканала  
**Тип:** Веб-приложение (SPA)  
**Хранение:** localStorage браузера  
**Целевая аудитория:** сотрудники водоканала, HR-отдел, администраторы

---

## 2. Функциональные требования

### 2.1. Аутентификация и авторизация

- **Регистрация пользователей:**
  - ФИО (обязательно)
  - Email (обязательно, уникально)
  - Пароль (минимум 6 символов)
  - Должность (выбор из списка)
  - Подразделение/цех
  - Дата приёма на работу

- **Вход в систему:**
  - Email + пароль
  - Запоминание сессии (localStorage)
  - Выход из системы

- **Роли пользователей:**
  - `admin` - полный доступ
  - `employee` - прохождение тестов
  - `hr` - просмотр результатов

### 2.2. Админ-панель

- **Управление пользователями:**
  - Список всех сотрудников
  - Добавление сотрудника вручную
  - Редактирование данных сотрудника
  - Блокировка/разблокировка
  - Удаление сотрудника
  - Сброс пароля

- **Управление тестами:**
  - Создание теста (название, описание, категория, время на прохождение)
  - Редактирование теста
  - Удаление теста
  - Клонирование теста
  - Публикация/скрытие теста

- **Управление вопросами:**
  - Добавление вопроса в тест
  - Типы вопросов:
    - Один правильный ответ (radio)
    - Несколько правильных ответов (checkbox)
    - Текстовый ответ
    - Вопрос на последовательность
  - Редактирование вопроса
  - Удаление вопроса
  - Импорт/экспорт вопросов (JSON)

- **Назначение тестов:**
  - Назначить тест конкретному сотруднику
  - Назначить тест подразделению
  - Установить срок сдачи
  - Установить количество попыток

### 2.3. Кабинет сотрудника

- **Мои тесты:**
  - Список назначенных тестов
  - Статус (не начат, в процессе, просрочен, пройден)
  - Оценка и дата прохождения
  - Начать/продолжить тест

- **Прохождение теста:**
  - Таймер обратного отсчёта
  - Навигация по вопросам
  - Пометка вопросов для проверки
  - Отправка ответов
  - Автосохранение прогресса

- **Результаты:**
  - Баллы/процент правильных ответов
  - Оценка (2, 3, 4, 5 или А/B/C/D)
  - Время прохождения
  - Детальный разбор ответов
  - История попыток

### 2.4. HR-панель (дополнительно)

- **Статистика по подразделениям:**
  - Средний балл
  - Процент прохождения
  - Просроченные тесты

- **Отчёты:**
  - Экспорт в Excel/CSV
  - Печать результатов

### 2.5. Категории тестов (для водоканала)

- Техника безопасности (ОТ и ТБ)
- Технологические процессы
- Сантехническое оборудование
- Водоснабжение и водоотведение
- Нормативные документы
- Экология и охрана окружающей среды
- Электробезопасность
- Оказание первой помощи
- Пожарная безопасность

---

## 3. Структура данных (localStorage)

```javascript
// Ключи хранилища:
- 'wt_users'          // Массив пользователей
- 'wt_tests'          // Массив тестов
- 'wt_questions'      // Массив вопросов
- 'wt_results'        // Массив результатов
- 'wt_assignments'    // Назначения тестов
- 'wt_currentUser'    // Текущий авторизованный пользователь
- 'wt_settings'       // Настройки системы
```

---

## 4. Технические требования

- **Frontend:** HTML5, CSS3, JavaScript (ES6+)
- **Стили:** Адаптивный дизайн, чистый UI
- **Без внешних зависимостей** (ванильный JS)
- **Совместимость:** Chrome, Firefox, Edge

## 5. Критерии приёмки

1. ✅ Регистрация и вход работают корректно
2. ✅ Админ может создавать тесты и вопросы
3. ✅ Админ может управлять пользователями
4. ✅ Сотрудник может проходить назначенные тесты
5. ✅ Результаты сохраняются и отображаются
6. ✅ Данные сохраняются в localStorage
7. ✅ Интерфейс адаптивный и понятный
